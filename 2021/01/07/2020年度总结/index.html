2020年度总结

发表于2021-01-08|更新于2021-02-15|无类可分
|字数总计:1.8k|阅读时长:5分钟|阅读量:

开篇

刚刚结束了地狱级别难度的期末考试，终于有时间来写这篇总结（不是真的想鸽）。

为什么之前没有写？

那是因为自己真的没什么可以说的（其实是太混了，虽然现在也很混）。

疫情

2020年一开头就不太平，爆发了疫情。新冠的爆发让我被迫在家呆了好几个月，我到现在都记得刚刚爆发时的情景，第一次感觉到生命是如此的脆弱。

不仅如此，新冠的爆发也让我看到了很多魔幻的事情，网上的事情我就不说了，大家懂的都懂，说说我碰上的。

稍微了解我的都知道我家是开药店的，疫情的爆发我相对很多人都更有体会，全镇都勒令关门禁止营业，除了药店，因为还要提供一些药物和口罩。虽说我们这边没有病例，可依旧是一件冒风险的事情。即使这样还会有人向政府举报我们家高价卖口罩（经过政府调查其实并没有），真的是让人心寒。

只能说医疗行业真的不容易。这其实也是为什么全家都叫我学医我还是选择学了计算机的原因（虽然最后进了数学院，有点遗憾）。

到现在疫情都还没有完全结束，总有死灰复燃的苗头，不过还好已经好转很多了，相信一切都会好起来的。

游戏

PC端

PC端我就只玩一款游戏，那就是LOL。这款游戏从我小学六年级一直陪伴我到现在，想想也是玩了好多年了。

不过今年玩的也少很多了，一部分是因为学习上的压力，另一方面就是网络风气。LOL的风气懂得都懂，其实每个游戏都这样，不过这游戏玩家多，现象超级明显，对新人极不友好。

移动端

移动端我就玩的还算多了，毕竟比较方便。

不过也只是玩了两款，那就是明日方舟和公主连结。

不过现在方舟也不玩了，虽然是一个好游戏，但是随着我学习上压力的递增，我也没有这么多时间来去通关了（想到第一次的危机合约，能玩上一天），到后面就都是抄作业了，时间久了就感受不到方舟的乐趣了，我也没有精力一起玩两款游戏，于是我就卸了（所以也没有截图了）。

说到公主连结就精神了，这游戏更加休闲，纯属一个养花瓶游戏，每天十分钟完成日常就可以了，十分适合我这些懒得玩的人，而且角色多好看啊（看看这个万圣忍，这不可爱吗？）。

这游戏最大的内容就是公会战（虽然公会战总是在摸鱼，但有时间还是会很努力的打的），因此我还认识了很多的志同道合的朋友，十分感谢各位的陪伴。

学习

专业学习

因为精力总不花在专业学习上，导致学分绩也就排在中游。（老混子了）

身为数学院的学子没学好数学还是挺惭愧的，以后有空研究深度学习了在认真看一下。（老师也都是研究深度学习的，开发在数学院地位低下hh）

开发

这方面其实也没什么说的了，直接看github吧（不要问什么有两段时间空了，在家是真的无法学习，以后要好好改正了）。

20年也没什么成就，花了挺多时间在这个博客上了。

我从大一下学期就接触了hexo，也就那时候就搭建好了这个博客，当时还是用的是next主题，什么都没整明白。

不过虽然搭建好了，但是没怎么管理，文章也就没怎么写，当时是觉得反正没什么人看，没怎么有动力弄得更好。

今年我就改变了这个想法。毕竟这写的文章不给别人看也是要给自己看的，所以要尽量写好，而且弄得更顺眼。

搭建这个博客的确是很能提高能力的一件事情，当然是尽量完善之后。简单说说搭建这个博客学到了什么。

  1. hexo与github pages（coding等）
  2. 域名解析，也因此买了自己的域名，并备了案
  3. html + css + javascript三件套，算是加深一点印像吧
  4. github action等一系列技术
  5. 使用CDN来加速自己的网站，怎样进行云存储
  6. 很多网络和排错的知识，比如Chrome的F12中的Network。
  7. Linux基本命令和使用一些基本的服务器软件，比如什么Nginx等

不得不说开发真的是一步步上云，上云是必然的趋势了。

读书方面

20年下半年我又重新拾起了书本，因为之前我实在看不下书，一看就困。但是书上会有很多知识比网上的资料要详细（因为又很多书都很经典），只有读书才能学到更多的东西，有很多书还是很有趣的。让我想想我20年看过什么书（当然除去课本）。

  • 网络是怎样连接的
  • C++ Primer Plus
  • 鸟哥的私房菜
  • 小灰的算法之旅
  • Java并发编程艺术
  • 大话数据结构
  • 哈喽算法

说了这么多其实有些没完全读完，就别如那本C++ Primer Plus，有这本书的都懂，而且我学的Java，有时间在看吧。

毕竟是从这个学期才开始认真好好读点书，所以看起来很少，21年尽量会看多一点的。

身体

啊这 。这个其实也就老样子了，依旧的不健康作息。

特别是前两三周，每天复习到凌晨，以后需要改了。

还有就是运动方面，不是不想跑步，因为疫情，好不容易来到学校前期又太忙后期风吹的我脑袋疼（找借口还是有一手的）。

不行了，真的要好好锻炼了，不然整天都是没精神的。

总结

20年懒散了，21年就必须更加努力才行了。说到努力，其实看到很多网上和现实的大佬，真的是强的人还很努力，对比下自己就完爆了。

的确人不能一直对一件事情保持热情（可能只是我不能吧），所以总会有些时候会丧失对学习的热情，总会颓废几天，不过之后还是要努力学习啊。

想到了DIYgod的签名，写代码是热爱，写到世界充满爱。这可能就是强者吧，越学习越热爱，我什么时候也能这么强。

emm,不说这么多了，21年也希望各位能变得更好吧，毕竟21年对于我们这一届来说还是很重要的，送各位一句话，也是小米的标语，永远相信美好的事情即将发生。
